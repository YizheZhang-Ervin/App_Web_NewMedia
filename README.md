# New Media

## Technology Stack

- FrontEnd: Vue + Vue Router + ElementUI + Axios + Unity(WebGL)
- BackEnd: SpringBoot + Actuator/Prometheus/JMX/Druid(暂无) + MySql/MyBatis

## Build FrontEnd

- vue create xxproject
- vue add router
- vue add element
- npm run build
- npm run serve

## Run BackEnd

- java -jar newmedia-spring-0.0.1-SNAPSHOT.jar
